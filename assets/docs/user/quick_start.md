# Quick Start

Last updated: 2026-06-02

## What The Application Does

AEGIS Geospatial View is a chat-driven geospatial workspace. The user describes a location and requested action in plain language, and the system returns map-focused results with session context and overlays when applicable.

## Primary Screens

- Workspace (`/`)
- Geodata (`/geodata`)
- Access configurations (`/access-configurations`)
- Model settings (`/settings`)

## Quick Start Journey

1. Open the app.
2. Ask for a place or coordinates and what you want to see.
3. Wait for the assistant response and map session.
4. Review the output and overlay controls.
5. Use the Operations Bar to move to geodata, access configuration, or model settings.
